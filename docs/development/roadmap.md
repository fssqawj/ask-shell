# Roadmap

Future plans and features for Ask-Shell.

## Version 0.3.0 (Q1 2026)

### Multi-LLM Support

- [ ] Claude integration
- [ ] Gemini support
- [ ] Ollama local models
- [ ] Provider plugin system

### Enhanced Safety

- [ ] Configurable safety levels
- [ ] Custom blacklist patterns
- [ ] Sandbox execution mode
- [ ] Undo/rollback capability

### Usability Improvements

- [ ] Command history with search
- [ ] Task templates library
- [ ] Auto-completion in interactive mode
- [ ] Colored diff preview for file changes

## Version 0.3.0 (Q2 2026)

### Collaboration Features

- [ ] Task sharing and replay
- [ ] Team templates
- [ ] Execution logs export
- [ ] Integration with CI/CD

### Advanced Execution

- [ ] Parallel command execution
- [ ] Remote server support (SSH)
- [ ] Docker container execution
- [ ] Kubernetes integration

### Developer Experience

- [ ] VS Code extension
- [ ] Web UI interface
- [ ] REST API server mode
- [ ] Python SDK for embedding

## Version 0.4.0 (Q3 2026)

### Intelligence Upgrades

- [ ] Custom command aliases

### Platform Expansion

- [ ] Windows PowerShell optimization

- [ ] Nushell integration
- [ ] Cloud shell environments

### Enterprise Features

- [ ] Compliance policies
