# Changelog

All notable changes to this project will be documented in this file.

## [0.3.0] - 2026-01-29

### Added
- Enhanced skill system with Browser, PPT, and Image skills
- Browser automation capabilities using Playwright
- PPT generation from natural language
- Image generation capabilities
- Extensible skill architecture
- Persistent browser sessions for multi-step web tasks

### Changed
- Refactored agent to use skill-based architecture
- Improved command execution safety with dual-layer protection
- Enhanced documentation reflecting skill-based system
- Updated README files with new features and capabilities
- Improved error recovery and retry mechanisms

### Fixed
- Various stability improvements for long-running tasks
- Better handling of edge cases in command execution
- Improved context management between task steps

## [0.2.0] - 2025-12-xx

### Added
- Initial release with core ask-shell functionality
- Natural language to shell command translation
- Interactive mode support
- Safety checks for dangerous operations
